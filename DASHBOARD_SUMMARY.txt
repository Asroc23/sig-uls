API ENDPOINTS - DETAILED
================================================================================

ENDPOINT 1: GET /api/graduates/by-year
  Description: Groups graduates by graduation year
  Query Params (optional):
    - gender: 'male' or 'female'
    - career_id: numeric ID
  Response: { labels: ["2023", "2024"], data: [5, 3] }
  Database: SELECT graduation_year, COUNT(*) FROM graduates GROUP BY year

ENDPOINT 2: GET /api/graduates/by-gender  
  Description: Groups graduates by gender
  Query Params (optional):
    - graduation_year: numeric year
    - career_id: numeric ID
  Response: { labels: ["Masculino", "Femenino"], data: [10, 8] }
  Database: SELECT gender, COUNT(*) FROM graduates GROUP BY gender

ENDPOINT 3: GET /api/graduates/by-career
  Description: Groups graduates by career, sorted by count descending
  Query Params (optional):
    - graduation_year: numeric year
    - gender: 'male' or 'female'
  Response: { labels: ["Engineering", "Business"], data: [15, 3] }
  Database: SELECT c.name, COUNT(g.id) FROM graduates g 
            JOIN careers c ON g.career_id = c.id 
            GROUP BY c.id, c.name ORDER BY COUNT DESC

================================================================================
DATABASE QUERIES
================================================================================

By Year Query (with filters):
  SELECT graduation_year, COUNT(*) as total 
  FROM graduates
  WHERE (gender = 'male' OR gender IS NULL)
  AND (career_id = 5 OR career_id IS NULL)
  GROUP BY graduation_year
  ORDER BY graduation_year

By Gender Query (with filters):
  SELECT gender, COUNT(*) as total
  FROM graduates
  WHERE (graduation_year = 2023 OR graduation_year IS NULL)
  AND (career_id = 5 OR career_id IS NULL)
  GROUP BY gender

By Career Query (with filters, joins careers):
  SELECT careers.name, COUNT(graduates.id) as total
  FROM graduates
  JOIN careers ON graduates.career_id = careers.id
  WHERE (graduation_year = 2023 OR graduation_year IS NULL)
  AND (gender = 'male' OR gender IS NULL)
  GROUP BY careers.id, careers.name
  ORDER BY total DESC

TROUBLESHOOTING GUIDE
================================================================================

Problem: "Charts not displaying"
Solution:
  1. Open browser F12 (Developer Tools)
  2. Check Network tab for API calls
  3. Verify API returns JSON: { labels: [], data: [] }
  4. Check Console for JavaScript errors
  5. Ensure you're logged in

Problem: "Filters not working"
Solution:
  1. Inspect Network tab for filter requests
  2. Verify query parameters are sent correctly
  3. Check API response format
  4. Look for console errors
  5. Try refreshing page

Problem: "Dark mode colors wrong"
Solution:
  1. Check HTML has 'dark' class
  2. Verify Tailwind CSS dark mode config
  3. Try toggling dark mode manually
  4. Clear browser cache
  5. Hard refresh (Ctrl+Shift+R)

Problem: "API returning 401/redirect"
Solution:
  1. Check if logged in (login page should appear)
  2. Verify auth middleware is working
  3. Check session is valid
  4. Clear cookies and re-login
